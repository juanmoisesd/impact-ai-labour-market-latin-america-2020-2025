# Reproducibility Guide

## Impact of Artificial Intelligence on the Labour Market in Latin America (2020-2025)

### Requirements
- Python 3.8+, pandas, matplotlib, seaborn, numpy

### Steps
1. `git clone https://github.com/juanmoisesd/impact-ai-labour-market-latin-america-2020-2025`
2. `pip install -r requirements.txt`
3. Open and run `analysis.ipynb`

### Data Source
Zenodo: https://doi.org/10.5281/zenodo.19211739
